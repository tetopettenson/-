■作品名

多言語同時翻訳ソフト

■このアプリケーションについて

入力欄に文字を入力したら翻訳ボタンを押すことで
アメリカ・イギリス英語と韓国語、中国語に同時翻訳できます。

■起動方法
「translation_app.exe」を実行してください。

■基本操作(デフォルト)
[キーボード]
特別な操作のキーはありません。
概要通りに操作してください。
